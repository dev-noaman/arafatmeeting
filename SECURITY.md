# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Mini-Meeting, please report it responsibly. **Do not create a public GitHub issue for security vulnerabilities.**

### How to Report

**Email**: Send details to [minimeetingapp@gmail.com](mailto:minimeetingapp@gmail.com)

### What to Include

- Description of the vulnerability
- Steps to reproduce the issue
- Potential impact and severity assessment
- Any suggested fixes (optional)
